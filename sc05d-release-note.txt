
  *** SC-05D Custom Kernel for Jellybean (AOSP) ***

  1. 内容
    CPU Freq
      384MHz ～ 1512MHz

    CPU Governor
      Performance
      PowerSave
      UserSpace
      Ondemand (default)
      Interactive (recommended)
      Conservative
      SmartassV2 (recommended)
      PegasusQ
      Sakuractive
      Wheatley

    I/O Scheduler
      cfq (default)
      bfq v4
      sio
      deadline
      noop
      vr
      row (recommended)

    Features
      TweakGS3 suport
        -- mDNIe control
        -- Vibrator control
      CPU Voltage Control support
      bootanimation support
        -> /system/media/bootanimation.zip or /data/local/bootanimation.zip
      bootsound support
        -> /system/media/bootsound.mp3
      bootmovie support
        -> /data/local/bootmovie.mp4
      external sd ext4 format support
      RAM Console support

    Others
      kernel version 3.0.8 -> 3.0.59
      Use compiler linaro 2012.12


  2. 注意事項
    使用した場合保証が切れます
    文鎮化の可能性があります


  3. 導入方法:
    SC05D-ICS-KBC-XXX-odin.tar.md5 : Odinで煮込んでください
    SC05D-ICS-KBC-XXX-sign.zip : CWM Recovery からinstallしてください


  4. その他
    何があっても自己責任でお楽しみください。
    また不具合等ありましたらご連絡ください。

    不意にリブートが発生した場合
    再起動後に/proc/last_kmsgを送ってくれると改善できる
    かもしれません。

    # cat /proc/last_kmsg > /sdcard/last_kmsg.log
